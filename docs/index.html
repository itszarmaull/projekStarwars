⭐ Star Wars Galaxy Explorer ⭐

Jelajahi Planet & Karakter Legendaris dari Universe Star Wars

Tentang Web Ini

Star Wars Galaxy Explorer adalah aplikasi web interaktif untuk menjelajahi berbagai planet dan karakter dari universe Star Wars. Data diambil dari API publik Star Wars (SWAPI) dan ditampilkan dengan tampilan modern dan animasi.

Teknologi yang Digunakan

  • HTML5 & CSS3
  • JavaScript (ES6+) & Axios
  • Tailwind CSS untuk styling modern
  • Three.js untuk animasi 3D latar belakang galaksi
  • SWAPI API untuk data Planet & Karakter
